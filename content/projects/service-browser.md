[Service Browser](https://play.google.com/store/apps/details?id=com.druk.servicebrowser) is an Android utility for discovering and inspecting network services via DNS-SD. It resolves IP addresses, ports, and TXT records, and lets developers register their own services. Built with pure Kotlin, Android's native NsdManager, and Material Design 3. The source code is available under the Apache 2.0 license.
