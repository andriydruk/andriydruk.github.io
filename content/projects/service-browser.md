<span style="color: var(--fallout-amber);">[PROJECT NAME]</span> Service Browser / DNSSD for Android

<span style="color: var(--fallout-amber);">[DESCRIPTION]</span> Network service discovery utility that discovers all services registered in a network and resolves meta-information about them.

<span style="color: var(--fallout-amber);">[CAPABILITIES]</span>
- Discover all network services automatically
- Resolve IP addresses and port information
- Register custom services in any domain (developer feature)
- Full network service meta-information display
